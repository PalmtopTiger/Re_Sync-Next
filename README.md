## Описание

Функционально идентична [Re_Sync](https://bitbucket.org/Lord_D/re_sync), за исключением:

- наличия интерфейса;
- отсутствия пакетной обработки;
- переписаных парсеров и фильтров;
- кривого сохранения графика;
- низкого качества кода.
